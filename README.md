# Light wizard

<a href ="https://abdullahhamdan.itch.io/light-wizard">Light Wizard</a> is a game created for Game Jam 2022.1, where the theme was "Not Real".
The game was developed using unity Engine, and assets were sourced from Brackeys.

## Features

- **Shooting**: Shoot enemies and take damage.
- **Audio**: Sound feedback for shooting, damage, and environment.
- **Graphics**: Stylized visuals built in Unity Engine.
- **Animations**: Character and combat animations.
- **Health System**: Player and enemy damage mechanics.

## Tech Stack

- **Engine:**: Unity engine
- **Assets:** Brackeys Asset Packs

## Getting started

- If you are running the game:

- Download or clone the repository.

- Open the project in Unity Engine.

- Click Play to run inside the editor
